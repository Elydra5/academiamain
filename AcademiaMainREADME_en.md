Documentation

When a user logs into the site, they are greeted by the login interface, where the following can be seen:

- Username - Here you need to enter the desired username
- Password - Here you need to enter the desired password
- Forgot password - Here you can change the password if the user has forgotten the previous one

After successful login, the home page welcomes us, where the following things are visible:

**Dashboard:**

Here you can see all the data available on the site in one place. The structure looks like this:

- Students
- Active groups
- Today's attendance
- Monthly revenue
- Team distribution
- Participant changes
- Monthly billing

**Students:**

- Here you can find the enrolled students, and you can also enroll new ones.
- There is also the possibility to modify or even delete student data if necessary.
- Once we have enrolled the students, the following can be seen - students' names, monograms, and phone numbers.

**Groups:**

- Here you can create groups for students, to which we can add them.

When creating a group, the following things need to be specified:

- Group name
- Short description
- Moodle id (numbers only)
- Closing date (optional)
- Teacher (selectable)
- Detailed description

**Users:**

- Here we see all users (teachers, students and admins)
- If there are none or you want to add more, then Add User

The following things need to be specified here:

- Username
- First name
- Last name
- Email
- Password
- What kind of permission you have on the site (teacher, student, admin)
- Moodle ID
- Activity

**Attendance:**

- Here you can add attendance.

Within this, the student can see:

- their name
- their group
- when it was added
- its duration
- teacher's name
- hourly rate

And they can issue a receipt for it.

**Billing:**

- student name
- number of hours studied
- total amount

Once these are available, the invoice can be issued.

**Calendar:**

On the calendar tab, you can view what planned or current events are on the dashboard. For example, what scheduled classes or ending courses there are. You can switch between monthly, weekly or daily views.
