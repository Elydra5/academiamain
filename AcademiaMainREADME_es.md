Documentación

Cuando un usuario inicia sesión en el sitio, la interfaz de inicio de sesión lo recibe, donde se pueden ver las siguientes cosas:

- Username - Aquí debe ingresar el nombre de usuario deseado
- Password - Aquí debe ingresar la contraseña deseada
- Forgot password - Aquí puede cambiar la contraseña si el usuario ha olvidado la anterior

Después de un inicio de sesión exitoso, la página de inicio nos recibe, donde se pueden ver las siguientes cosas:

**Dashboard:**

Aquí se pueden ver todos los datos disponibles en el sitio en un solo lugar. La estructura se ve así:

- Estudiantes
- Grupos activos
- Asistencia de hoy
- Ingresos mensuales
- Distribución del equipo
- Cambios de participantes
- Facturación mensual

**Students:**

- Aquí se encuentran los estudiantes inscritos, y también se pueden inscribir nuevos.
- También existe la posibilidad de modificar o incluso eliminar los datos de los estudiantes si es necesario.
- Una vez que hayamos inscrito a los estudiantes, se puede ver lo siguiente: nombres de los estudiantes, sus monogramas y números de teléfono.

**Groups:**

- Aquí se pueden crear grupos para los estudiantes, a los que podemos agregarlos.

Al crear un grupo, se deben especificar las siguientes cosas:

- Nombre del grupo
- Descripción breve
- ID de Moodle (solo números)
- Fecha de cierre (opcional)
- Profesor (seleccionable)
- Descripción detallada

**Users:**

- Aquí vemos todos los usuarios (profesores, estudiantes y administradores)
- Si no hay ninguno o desea agregar más, entonces Add User

Aquí se deben especificar las siguientes cosas:

- Nombre de usuario
- Nombre
- Apellido
- Correo electrónico
- Contraseña
- Qué tipo de permiso tiene en el sitio (profesor, estudiante, administrador)
- ID de Moodle
- Actividad

**Attendance:**

- Aquí se puede agregar asistencia.

Dentro de esto, el estudiante puede ver:

- su nombre
- su grupo
- cuándo se agregó
- su duración
- nombre del profesor
- tarifa por hora

Y puede emitir un recibo por ello.

**Billing:**

- nombre del estudiante
- número de horas estudiadas
- monto total

Una vez que estos estén disponibles, se puede emitir la factura.

**Calendario:**

En la pestaña del calendario, puede ver qué eventos planificados o actuales hay en el dashboard. Por ejemplo, qué clases programadas o cursos que terminan hay. Puede cambiar entre vistas mensuales, semanales o diarias.
